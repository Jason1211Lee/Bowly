# Bowly 快速啟動指南 🚀

## 一鍵啟動

### Windows 用戶

雙擊根目錄中的 `start.bat` 文件即可自動啟動應用！

```
Bowly/
└── start.bat  ← 雙擊這個文件
```

**首次運行**：會自動安裝依賴，請耐心等待

### macOS / Linux 用戶

在終端中運行以下命令：

```bash
chmod +x start.sh        # 首次執行需要給予執行權限
./start.sh
```

或直接使用 npm：

```bash
npm start
```

## 常見問題

### Q: 啟動時出現 "npm: 無法找到命令"

**A:** Node.js 未正確安裝。請訪問 https://nodejs.org/ 下載最新版本

### Q: 無法訪問 http://localhost:3000

**A:** 
1. 確認伺服器正在運行（見上方啟動輸出）
2. 檢查 3000 端口是否被佔用：
   
   **Windows**：
   ```powershell
   netstat -ano | findstr :3000
   ```
   
   **macOS/Linux**：
   ```bash
   lsof -i :3000
   ```

3. 修改端口：編輯 `.env` 文件
   ```
   PORT=8080  # 改成 8080 或其他空閒端口
   ```

### Q: 服務器啟動後立即關閉

**A:** 查看是否有錯誤信息：
```bash
npm start  # 直接在終端運行以看到完整錯誤
```

### Q: 如何修改啟動端口？

**A:** 編輯 `.env` 文件：
```bash
PORT=8000  # 改成想要的端口
```

然後重新啟動應用
